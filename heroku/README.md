Usage: Copy all files in this directory (make sure you include .godir) into a
new git repository. Add lmk.conf and rules.conf to this repository. Commit.
Follow the instructions [here](http://mmcgrana.github.io/2012/09/getting-started-with-go-on-heroku.html)
to finish installing on Heroku. Make sure the redistogo and scheduler addons
are enabled for your Heroku instance. Use the scheduler to run /poll as
frequently as you would like.
